| Код треб| Требование | Комментарий |
|---------|------------|-------------|
|SEC-01|Взаимодействие между системой и API Сервисов Умный дом должно выполняться схеме клиент-сервер по HTTP-протоколу (REST API).||
|SEC-02|Трафик между системой и API Сервисов Умный дом должен быть зашифрован.||
|SEC-03|При подключении к API Сервисов Умный дом должна проверяться аутентификация и авторизация учетных записей, под которыми выполняются запросы к API. Рекомендуется использовать OIDC-протокол.||
|SEC-04|Подключение к API Сервисов Умный дом должна быть разрешено только с определенных IP-адресов партнеров по интеграции.||
|SEC-05|При настройке прав доступа партнеров к разным эндпойнтам API и управляемым объектам должен соблюдаться принцип минимальных привилегий.||
|SEC-06|Должен вестись журнал аудита авторизаций и запросов к сервисам API. Доступ к журналам аудита должен быть у ограниченной группы администраторов сервиса.||
|SEC-07|При настройке прав доступа партнеров к разным эндпойнтам API и управляемым объектам должна использоваться модель ABAC для более гранулярного доступа.||
|SEC-08|Эндпойнт сервиса должен быть защищен системой Anti-DDoS.||
